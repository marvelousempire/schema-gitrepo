# Journal Update - Schema Gitrepo Origin

## Summary

Schema Gitrepo was created as the reusable blueprint for building future Git repositories.

It is the sibling schema to Schema Agent.

## Key Decision

Schema Agent defines how to build agents.

Schema Gitrepo defines how to structure repos.

- Root file standards
- Repo folder standards
- Feature ticket pattern
- Journal pattern
- Release ledger pattern
- Repo manifest pattern
- Templates
- New repo checklist
