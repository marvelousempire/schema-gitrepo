# Git Repo Schema

## Purpose

The Git Repo Schema is the reusable pattern for creating well-structured repos.

## Core Idea

A repo is a project memory system, not only a folder.

## Main Parts

- Purpose
- README
- ABOUT
- PRD
- Manifest
- Release ledger
- Feature ledger
- Tickets
- Docs
- Journal
- Templates
- Tests
