# FEAT-0006 - New Repo Checklist

## Status

Draft

## Valid Intent

Give builders a clear checklist for creating a new Git repo.

## User Value

A new repo can be created without forgetting important files, folders, or validation steps.

## Description

Create a checklist that covers root files, folders, ledgers, wiki notes, journals, and validation requirements.

## Acceptance Criteria

- Checklist exists.
- Required root files are listed.
- Required folders are listed.
- Validation criteria are listed.
