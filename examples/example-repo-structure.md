# Example Repo Structure

## Purpose

This file shows the standard Schema Gitrepo structure.

## Structure

```text
example-repo/
├── README.md
├── ABOUT.md
├── PRD.md
├── LICENSE
├── repo-manifest.md
├── release-ledger.md
├── docs/
├── features/
│   ├── README.md
│   ├── ledger.md
│   └── tickets/
├── obsidian/
│   ├── README.md
│   └── journal/
├── templates/
├── tests/
│   └── README.md
└── examples/
```

## Rule

A repo should be clear enough that another human or AI can enter it safely.
