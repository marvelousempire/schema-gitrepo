# Tests

## Purpose

This folder tracks validation notes for Schema Gitrepo.

## Validation Checklist

- README exists.
- ABOUT exists.
- PRD exists when needed.
- Repo manifest exists.
- Release ledger exists.
- Feature ledger exists.
- Core docs exist.
- Obsidian notes exist.
- Journal exists.
- Templates exist.
- Example structure exists.

## Rule

A repo schema is ready when another AI can read it and build from it.
