# Repo Manifest

## Purpose

The Repo Manifest lists the important files and folders in Schema Gitrepo.

## Root Files

| File | Purpose |
| --- | --- |
| README.md | Front door and quick start |
| ABOUT.md | Meaning and identity |
| PRD.md | Product requirements |
| LICENSE | License |
| repo-manifest.md | Repo inventory |
| release-ledger.md | Milestone memory |

## Folders

| Folder | Purpose |
| --- | --- |
| docs/ | Schema guides and build rules |
| features/ | Feature ledger and tickets |
| features/tickets/ | Small buildable feature files |
| obsidian/ | Wiki-style notes |
| obsidian/journal/ | Decision history |
| templates/ | Reusable repo templates |
| tests/ | Validation notes |
| examples/ | Example repo structures |

## House Rule

When a new important file or folder is added, update this manifest.
