# Release Ledger Pattern

## Purpose

The Release Ledger Pattern explains how important repo milestones are remembered.

## Core Rule

When the repo reaches an important state, update the release ledger.

## Suggested Columns

```text
Version
Type
Summary
Status
```

## Release Types

- concept
- docs
- feature
- template
- validation
- schema
- journal
- release

## Rule

The release ledger should help future readers know what changed and why.
