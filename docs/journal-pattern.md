# Journal Pattern

## Purpose

The Journal Pattern explains how a repo remembers important decisions.

## Core Rule

If a decision changes the project, write it down.

## Folder

```text
obsidian/journal/
```

## Journal Entries Should Capture

- What changed
- Why it matters
- Files affected
- Decisions made
- Next actions

## Rule

The journal should preserve context that might be lost in chats, commits, or memory.
